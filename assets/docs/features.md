# GreatFrameForge Features 🚀 (v1.05)

### 🎨 Advanced Compositing
- **Dual-Layer Engine:** Independent control over **Base Plate** and **Inset (PiP)** assets.
- **Visual Filters:** Per-layer **Contrast** and **Brightness** sliders for instant color grading.
- **Precision Transforms:** Full control over Pan, Zoom, Rotation, and Mirroring.
- **Dynamic Masking:** Toggle between **Rectangle** and **Circle** masks for inset images.
- **Pro Gradients:** Dual-layer linear gradient engine with custom rotation, spread, and opacity controls.

### ✍️ Next-Gen Typography (Text Forge)
- **Interactive Handles:** Direct on-canvas controls for **Character Spacing (Kerning)** and **Vertical Stretch**.
- **Custom Font Engine:** Load any `.ttf`, `.otf`, or `.woff` file from your local system instantly.
- **6-Layer Stack:** Reorganized UI with a primary 3-layer stack and a "More Text Layers" overflow for complex designs.
- **Visual FX:** Professional **Outline** and **Drop Shadow** controls with full color pickers.
- **Global Scaling:** Precision global XY positioning and layout scaling.

### 📏 YouTube Optimization
- **YT Safe Zones:** Toggleable, asymmetrical guides (80px Left, 180px Right).
- **Overlay Protection:** Dedicated visual indicators for the **Duration Counter**, **End Cards**, and **Hover Icons**.

### 💾 Preset & Export
- **Unified Preset Manager:** Sleek Load/Save/Delete interface with one-click overwrite capability.
- **JSON Portability:** Import and export your entire preset library for backup or sharing.
- **High-Res Engine:** Offscreen rendering for crisp **1080p** or **1440p** exports.

### 💻 Distribution & Security
- **Standalone Portable:** Package the entire app into a single, no-install `.exe` for USB or offline use.
- **Proprietary Protection:** Industry-standard **JavaScript Obfuscation** to scramble logic and prevent unauthorized code access.
- **Creator Signature:** Integrated "Crafted by Y0kenB" branding with social media connectivity.

### 💳 Tiered Accessibility
- **Free Tier:** All core transform, color picking, and slider functionalities remain fully unlocked for everyone. Capped layer counts (1 Inset, 1 Gradient, 1 Text field ) support entry-level composition.
- **Base Tier:** Unlocks unlimited layers, YouTube Safe Zones, unrestricted presets, and Export/Import functionalities to support professional workflows. text gradients, and text font selectors are base tier features.
